##

A modern and fully responsive admin dashboard built with React, Vite, and Material UI. Includes interactive charts, data grids, and calendar integration for managing and visualizing data efficiently.

## Features

Responsive and clean design

Interactive charts with Nivo

Calendar integration using FullCalendar

Data tables with MUI DataGrid

Easy form handling with React Hook Form

Smooth navigation using React Router

## Installation

1. npm install
2. npm run dev
